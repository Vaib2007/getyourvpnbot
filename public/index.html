Get Your VPN Bot

Welcome to the Surfshark VPN Bot! This Telegram bot helps you generate WireGuard VPN configuration files for your Surfshark subscription.

🔐
Secure WireGuard key generation
🌍
Access to all Surfshark server locations
⚡
Fast and automated configuration
📱
Works with any WireGuard-compatible app

How to Use

  1. Start a chat with the bot on Telegram
  2. Send /start to see available commands
  3. Use /connect email:password with your Surfshark credentials
  4. Download the generated WireGuard configuration file
  5. Import it into your WireGuard app and connect!
